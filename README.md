# 2.功能实现部分
## 2.1 登陆部分
- 两次MD5
```markdown
 1. 用户端： PASS = MD5( 明文 + 固定 Salt) 

 2. 服务端： PASS = MD5( 用户输入 + 随机 Salt)

	通过两次MD5，可以增大http明文传输过程或数据库被盗后，黑客通过彩虹表等手段反推出明文密码的难度（有一定作用，但不能保证绝对安全）。
```
添加的依赖如下：

``` javascript
        <dependency>

```
